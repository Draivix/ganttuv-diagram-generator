## [1.0.0] — 2026-04-27

### Added
- Initial public release of `ganttuv-diagram-vzor.xlsx`
- Sheet **„Ganttův diagram"** — task table + 26-week formula-driven timeline
- Sheet **„Vizualizace"** — day-resolution stacked bar chart simulation
- Sheet **„Návod"** — 8-step Czech usage guide
- Sheet **„O šabloně"** — branding, MIT license, AutoERP CTA
- 17 sample tasks for "Implementace ERP" project across 5 phases
- Status data validation with conditional formatting per state
- Color-scale conditional formatting on progress %
- Weighted project-progress summary row (SUMPRODUCT-based)
- Print-ready A4 landscape page setup
- TypeScript generator script in `scripts/generate.ts`
- Live web tool at <https://autoerp.cz/ganttuv-diagram-generator> with interactive Gantt builder
